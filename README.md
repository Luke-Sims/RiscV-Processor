# Intro
Ceci est l'état actuel de mon projet processeur RISC V. Il comporte actuellement les fonctionalitées jusqu'a l'écriture en mémoire et l'implémentation des branches est en cours
# Executer le code
Compilation: `make` puis `make run`
simulation: `make sim`

# Architecture du projet
- Build (Fichiers de compilation .o)
- Scripts (L'ensemble des fichiers .hex d'instructions)
- Sim (*.ghw)
- Source (fichiers principaux de fonctionnement du processeur)
- Testbench (fichiers de test)
